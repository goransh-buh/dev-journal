# Tool: pydantic

**Description:** data validation library  
**Date explored:** 2026-06-15

## Notes

Define schemas as Python classes. Automatic type coercion, validation errors, and JSON serialization.

## When to use it

Worth reaching for when the standard library solution is too slow,
too verbose, or missing important features for the use case.

## Setup

```bash
pip install pydantic
```
