# Tool: just

**Description:** command runner  
**Date explored:** 2026-06-12

## Notes

Like make but without the footguns. Justfile syntax is clean and cross-platform.

## When to use it

Worth reaching for when the standard library solution is too slow,
too verbose, or missing important features for the use case.

## Setup

```bash
pip install just
```
