# Tool: lazygit

**Description:** terminal UI for git  
**Date explored:** 2026-07-15

## Notes

Navigate commits, stage hunks, and manage branches without memorizing git commands.

## When to use it

Worth reaching for when the standard library solution is too slow,
too verbose, or missing important features for the use case.

## Setup

```bash
pip install lazygit
```
