# Tool: httpx

**Description:** async HTTP client for Python  
**Date explored:** 2026-07-12

## Notes

Drop-in replacement for requests with native async support. Connection pooling built in.

## When to use it

Worth reaching for when the standard library solution is too slow,
too verbose, or missing important features for the use case.

## Setup

```bash
pip install httpx
```
