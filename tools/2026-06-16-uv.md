# Tool: uv

**Description:** Python package manager written in Rust  
**Date explored:** 2026-06-16

## Notes

Replaces pip + venv. `uv sync` installs from lockfile in under a second.

## When to use it

Worth reaching for when the standard library solution is too slow,
too verbose, or missing important features for the use case.

## Setup

```bash
pip install uv
```
