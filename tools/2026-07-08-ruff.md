# Tool: ruff

**Description:** Python linter written in Rust  
**Date explored:** 2026-07-08

## Notes

100x faster than flake8. Single tool replaces flake8 + isort + pyupgrade.

## When to use it

Worth reaching for when the standard library solution is too slow,
too verbose, or missing important features for the use case.

## Setup

```bash
pip install ruff
```
