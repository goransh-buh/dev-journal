# Tool: ripgrep

**Description:** fast grep replacement  
**Date explored:** 2026-06-17

## Notes

`rg <pattern>` respects .gitignore, handles binary files, and is significantly faster than grep.

## When to use it

Worth reaching for when the standard library solution is too slow,
too verbose, or missing important features for the use case.

## Setup

```bash
pip install ripgrep
```
