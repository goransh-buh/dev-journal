# Tool: mermaid

**Description:** diagramming in Markdown  
**Date explored:** 2026-07-16

## Notes

Write flowcharts and sequence diagrams as text; renders in GitHub READMEs automatically.

## When to use it

Worth reaching for when the standard library solution is too slow,
too verbose, or missing important features for the use case.

## Setup

```bash
pip install mermaid
```
