# TIL — Git (2026-06-21)

## What I learned

`git rebase -i HEAD~3` lets you squash, reorder, or reword the last 3 commits before pushing.

## Why it matters

Understanding this helps write more correct and efficient code.
Small details like this compound over time into significantly better intuitions.
