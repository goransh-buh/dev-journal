# Problem: Detect a cycle in a directed graph.

**Category:** Graph  
**Date:** 2026-07-21

## Problem Statement

Detect a cycle in a directed graph.

## Approach

DFS with three states: unvisited, in-stack, done. A cycle exists if we reach an in-stack node.

## Complexity

- Time: O(n log n) in the general case
- Space: O(n) auxiliary

## Key Insight

Breaking the problem into smaller subproblems and recognising the pattern
is more valuable than memorising the solution.
